Zwei-Faktor-Authentifizierung
Die Zwei-Faktor-Authentifizierung bietet zusätzliche Sicherheit für Webmin-Konten. Ist sie aktiviert, muss der Benutzer ein einmaliges Authentifizierungstoken eingeben, das in der Regel von einer App, einem Hardware-OTP-Gerät oder per SMS generiert wird.

Auf dieser Seite können Sie einen Authentifizierungsanbieter für Webmin-Benutzer auf Ihrem System auswählen. Nach der Aktivierung kann sich jeder Benutzer im Modul „Webmin-Benutzer“ für die Zwei-Faktor-Authentifizierung anmelden. Die verfügbaren Zwei-Faktor-Anbieter sind:

  • TOTP-Authentifikator
    Dies ist eine Desktop- oder Smartphone-App, die das Standard-TOTP-Protokoll implementiert. Jeder Benutzer muss mit der App einen QR-Code scannen, um seine Token mit dem Webmin-Server zu verknüpfen.
  • Authy
    Es handelt sich um einen kommerziellen Dienst, der eine Smartphone-App und eine Website zur Verwaltung von Zwei-Faktor-Benutzern bereitstellt. Jeder Benutzer muss auf der Authy-Website ein Konto erstellen und es mit seinem Smartphone verknüpfen.
